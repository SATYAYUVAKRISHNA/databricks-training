# SQL Assignments

This repository contains the SQL assignments and practice queries that I completed using MySQL and VS Code.  
The assignments cover different SQL concepts from basic to intermediate level with sample tables, insert statements, and query solutions.

I organized the repository topic-wise so it is easy to understand and execute each assignment separately.

## Topics Included

- String Functions
- Number Functions
- NULL Functions
- Date & Timestamp Functions
- REGEX Functions
- Joins
- Window Functions
